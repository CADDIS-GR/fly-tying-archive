---

## 🎯 컬렉션 개요

이 컬렉션은 3가지 버전의 크로노미드 패턴으로 구성되어 있으며, 각각 다른 상황과 조건에 최적화되어 있습니다. **Bubble Dot** 코어 기술을 활용하여 유충이 부상할 때 머금는 공기 방울을 완벽하게 재현했습니다.

### 📦 구성품
- **Pink Larva 3D** - 입체적 반사 버전
- **Pink Larva 2D** - 슬림 실루엣 버전
- **White Pupa** - 순수 화이트 버전

---

## 🔬 코어 기술: Bubble Dot

바니시가 중력을 따라 비드 아래로 흐르며 형성되는 미세한 방울이 핵심입니다.

### 기술 원리
```
1. Thin UV Resin 도포
2. 비드 하단으로 자연스럽게 흐름
3. Vanish Coat로 고정
4. Instant Curing으로 마무리
```

### 효과
- 🫧 유충의 공기 방울 완벽 재현
- 💎 빛 반사 극대화
- 🎯 송어의 시각적 트리거 자극

## 📋 패턴 상세

### 1️⃣ Pink Larva 3D

![Pink Larva 3D](https://raw.githubusercontent.com/CADDIS-GR/fly-tying-archive/main/images/PinkLarvar_3D.jpg)

**"입체적 반사, 공격성을 극대화하다"**

#### 재료
| 구분 | 재료 |
|------|------|
| **Hook** | DAIICHI 1760 #14 |
| **Under Thread** | UTC FL Pink 70D |
| **Body Material** | UTC FL Pink 70D |
| **Ribbing** | Red wire (BR) - 한 사이즈 굵게 |
| **Finish** | Thin UV Resin + Vanish Coat |

#### 특징
- 한 사이즈 굵은 wire를 사용하여 내부의 레드 와이어와 핑크 바디가 입체적으로 굴절
- 깊은 수심에서도 강렬한 시각적 자극
- 3D 효과로 공격 본능 유발

#### 최적 사용 조건
- ☀️ **날씨:** 한겨울 물색 맑고 햇살 좋은 날
- 🌊 **수심:** 3-6미터
- ⏰ **시간:** 오전 10시 - 오후 2시 (햇살 강할 때)
- 🌡️ **수온:** 4-8°C

## 📖 Master's Field Note

### 🎯 Sizing (사이즈 선택)
> "작을수록 송어는 부담 없이 삼키고 돌아섭니다."

- #14 사이즈가 입질 빈도를 확실히 높입니다
- 송어가 경계심을 낮추고 자연스럽게 접근
- 작은 사이즈일수록 거부 입질이 적음

### 📏 Depth (수심 공략)
> "카운트다운을 통해 송어가 유영하는 정확한 수심을 찾는 것이 먼저입니다."

**카운트다운 방법:**
```
1. 캐스팅 후 라인이 가라앉는 속도 측정
2. 1초당 약 30cm 침강 가정
3. 3-4-5-6미터 수심을 각각 테스트
4. 입질이 있는 수심을 기억하고 집중 공략
```

### 🎣 Action (액션 운용)
> "자연스러운 '스위밍' 동작으로 아주 느리게 유혹하는 것이 핵심입니다."

**추천 리트리브:**
1. **초저속 스위밍** - 5-10cm/초 속도
2. **긴 폴즈** - 3-5초 멈춤
3. **미세한 트위치** - 손목만 사용한 작은 움직임
4. **자연스러운 드리프트** - 물 흐름에 맡기기

## 🎨 타잉 팁

### 준비 단계
1. **훅 세팅**
   - DAIICHI 1760 #14를 바이스에 고정
   - 비드를 먼저 끼우기 (White 또는 텅스텐)

2. **언더레이어 구성**
   - 밑실을 훅 전체에 균일하게 감기
   - 컬러 톤의 70%는 밑실이 결정

3. **와이어 감기**
   - Pink Larva 3D: 한 사이즈 굵은 Red wire
   - Pink Larva 2D: 얇은 Flat Red Wire (XS)
   - 균일한 간격으로 감아 세그먼트 표현

4. **바디 형성**
   - 얇게 여러 번 감아서 반투명하게
   - 두껍게 감으면 무거워져서 액션 저하

5. **UV 레진 마무리**
   - Thin UV Resin을 얇게 도포
   - 비드 하단으로 흐르게 하여 Bubble Dot 형성
   - Vanish Coat로 한 번 더 코팅
   - UV 램프로 즉시 경화
